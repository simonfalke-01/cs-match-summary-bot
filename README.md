# Discord Bot

A bare minimum Discord bot written in Go.

## Setup

1. Install dependencies:
```bash
go mod tidy
```

2. Set your Discord bot token as an environment variable:
```bash
export DISCORD_BOT_TOKEN=your_bot_token_here
```

3. Run the bot:
```bash
go run main.go
```

## Features

- Responds to "ping" with "Pong!"
